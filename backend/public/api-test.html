Deadline Shield backend

If you see this page, the API server is running.

Use http:// not https:// — there is no SSL on port 5000. If the browser shows “error loading” or a security warning, check the address bar starts with http://localhost:5000.

Quick checks:

  • /api/health — JSON: is MongoDB connected?
  • /api — JSON list of base routes

Start the server from the backend folder: npm run dev
